Notes sur OpenSUSE Leap
=======================

Démarrage : appuyer sur F2 pour définir la langue

Rôle système : Personnalisé

Cliquer sur Système de base amélioré, ce qui décoche tout sauf Système de base.


Partitionnement
---------------

Si le disque est déjà partitionné en MBR, l'installateur mettra une table de
partitions MBR. En revanche, en l'absence de table de partitions, on aura une
table GPT. On devra obligatoirement créer une partition BIOS boot d'une taille
de 8 Mo au début du disque. 

La meilleure solution consiste à booter sur le DVD en mode secours et à zapper
la table de partitions existante avec gdisk > x > z. 

Schéma de partitionnement automatique par défaut :

  linux-q07j:~ # fdisk -l /dev/sda
  Disque /dev/sda : 40 GiB, 42949672960 octets, 83886080 secteurs
  Unités : secteur de 1 × 512 = 512 octets
  Taille de secteur (logique / physique) : 512 octets / 512 octets
  taille d'E/S (minimale / optimale) : 512 octets / 512 octets
  Type d'étiquette de disque : gpt
  Identifiant de disque : 4DACF3CA-C962-437D-B72C-1E65B26F4642

  Périphérique    Début      Fin Secteurs Taille Type
  /dev/sda1        2048    18431    16384     8M Amorçage BIOS
  /dev/sda2       18432 52017151 51998720  24,8G Système de fichiers Linux
  /dev/sda3    52017152 79689727 27672576  13,2G Système de fichiers Linux
  /dev/sda4    79689728 83886046  4196319     2G Partition d'échange Linux

Une partition BIOS boot, une partition principale, une partition /home et une
swap à la fin du disque.

Problèmes de Btrfs. On va donc opter pour ext4 au lieu de Btrfs et ne pas faire
de partition /home à part. 

La partition BIOS boot sert à amorcer un disque GPT depuis un BIOS
traditionnel.

Partitionnement en mode expert > Démarrer avec la proposition actuelle.

Cliquer sur le disque (sda) à gauche.

Supprimer l'une après l'autre les partitions prédéfinies.

Partition BIOS boot :

Ajouter > Taille personnalisée : 8 MiB

Rôle : Raw Volume (unformatted)

Ne pas formater le périphérique (laisser tel quel)

Partition ID : Partition de démarrage du BIOS

Partition /boot :

Ajouter > Taille personnalisée : 500 MiB

Rôle : Operating System

Système de fichiers : ext2

Point de montage : /boot

Partition / :

Ajouter > Taille personnalisée = Taille maximale - taille de la swap

Rôle : Operating System

Système de fichiers : ext4

Point de montage : /

Partition swap : 

Ajouter > Taille maximale (restant)

Rôle : swap
